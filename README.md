# 🏥 Hospital Emergency Room Dashboard

An interactive dashboard to analyze ER patient flow, wait times, and hospital efficiency.

## 📊 Overview

Emergency rooms often face challenges like long wait times and inefficient patient handling.  
This project analyzes ER data to uncover patterns and improve hospital decision-making.

## 🎯 Objectives

- Analyze patient wait times
- Identify peak congestion hours
- Study patient demographics
- Evaluate hospital performance

- ## 🛠 Tech Stack

- Power BI
- Python (for data cleaning)
- SQL (for querying)
- Excel / CSV dataset
